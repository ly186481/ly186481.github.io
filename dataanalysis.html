What is Data Analysis?

  • Data is a collection of facts.
  • Data Analysis is the collection,transforamtion and organization of data in order to draw conclusions, make predictions, and drive informed decision-making.

Analytical Thinking

Gap Analysis

  6. Destroy

Six phrases of Data Analysis

Ask - Prepare - Process - Analyze - Share - Act

  1. Ask : Define problem to be solved, make sure fully understand stakeholder expectation
  2. Prepare : Collect & Store data for process
  3. Process : Find and eliminate any error and unaccuracy to the result
  4. Analyze : Tools to transforms and organize the information to predict or useful information
  5. Share : Interpret result and share with others to help stakeholder make effective decisions
  6. Act : Puts to action to solve business problem
