naive Bayes, uses data about prior events to estimate the probability of future event. Classifiers based on Bayesian methods utilize training data to calculate an obeserved probability of each class based on feature values.

  • Text classification, like spam filtering, author identification, topic categorization.
  • Intrusion detection or anomaly detection in computer networks.
  • Diagnosing medical conditions, when goven a set of observed symptoms.
