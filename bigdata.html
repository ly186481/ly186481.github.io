Big Data

4 Dimensions

  • Volume : Large Amount of Data 41% growth rate for 2009-2020
  • Velocity: Speed of Data, You-tube, blog, Twitter, Facebook, Smart Phone e-mail, Google Search, on-line Trading
  • Variety: Diverse set of Data 80% are unstructured, video, audio, text
  • Veracity: Trustworthiness

    • Data Mining

    Data mining is the process of discovering patterns in large data sets involving methods at the intersection of machine learning, statistics, and database systems. It focuses on the discovery of unknown properties in the data. It uses many machine learning methods but with different goals. It is also employs as "Unsupervised Learning"

    • Data mining came from Statistic and AI
      1. Statistics involves formal modeling of underlying process generating data
      2. Smart statistics accurately identifies relationships
      3. Artificial Intelligence provides efficient methods to surf the data
      4. Smart algorithms utilize results to improve decision making
    • Data Mining task is the semi-automatic or automatic analysis to extract previously unknown, interesting patterns such as groups of data records (cluster analysis), unusual records (anomaly detection), and dependencies (association rule mining, sequential pattern mining)

    Artificial Intelligence

    Intelligence demonstrated by machines, machine intelligence

    • Marvin Minsky named AI(Dartmouth Conference 1956)
      1. 1990s Machine Learning Trend shifts from a knowledge-driven approach to a data-driven approach. Support vector machines (SVMs) and recurrent neural networks (RNNs) become popular. The fields of computational complexity via neural networks and super-Turing computation started.
      2. 2000s Support Vector Clustering and other Kernel methods and unsupervised machine learning methods become widespread
      3. 2010s Deep learning becomes feasible, which leads to machine learning becoming integral to many widely used software services and applications.

    Supervised Learning : Predictive Analytics

    Given training data, learn a model to predict Y(label) from X(features)

    • Classification
    • Estimation/Prediction
    Learning with a labeled training set. Typical for regression and classification-type tasks, when training data is readily available for computing errors and adapting the models

    Unsupervised Learning : Descriptive Analytics

    Given Data (features only), identify underlying patterns or structures in data.

    • Density estimation/Clustering
    • Structural Description and Inference Engine
    • Affinity Grouping
    Discover patterns in unlabeled data (self-organization) Typical for clustering and density estimation (the “no-output” tasks)

    Reinforcement Learning : Training data S-A-R (S: State, A:Action, R:Reward)

    • Develop an optimal policy (sequence of decision rule) for acting to maximize long-term feedback/reward
    • by experimentation, trial-and-error, with delayed rewards
    • when training data is not readily available or in short supply (often in Supervised Learning)
    Learn to act based on feedback/reward (by experimentation, trial-and-error, with delayed rewards). typical for control and optimization problems, also in context of classification and regression when training data is not readily available or in short supply (often in Supervised learning)
